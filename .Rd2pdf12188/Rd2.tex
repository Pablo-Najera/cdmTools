\item[Type]\AsIs{Package}
\item[Title]\AsIs{Useful Tools for Cognitive Diagnosis Modeling}
\item[Version]\AsIs{1.1.0}
\item[Date]\AsIs{2022-03-21}
\item[Description]\AsIs{Provides useful tools for cognitive diagnosis modeling (CDM). The packages includes the discrete factor loading method for Q-matrix estimation (Wang, Song, & Ding, 2018, <}\Rhref{https://doi.org/10.1007/978-3-319-77249-3_29}{doi:10.1007/978-3-319-77249-3\_29}\AsIs{>) and the Hull method for Q-matrix validation (Nájera, Sorrel, de la Torre, & Abad, 2021, <}\Rhref{https://doi.org/10.1111/bmsp.12228}{doi:10.1111/bmsp.12228}\AsIs{>). It also provides dimensionality assessment procedures for determining the number of attributes underlying CDM data, including parallel analysis and automated CDM fit comparison as explored in Nájera, Abad, and Sorrel (2021, <}\Rhref{https://doi.org/10.3389/fpsyg.2021.614470}{doi:10.3389/fpsyg.2021.614470}\AsIs{>). Lastly, the package provides some useful functions for CDM simulation studies, such as random Q-matrix generation and detection of complete/identified Q-matrices.}
